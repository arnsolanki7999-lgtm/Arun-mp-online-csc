# Arun MP Online CSC - इंदौर सरकारी सेवाएं गाइड

## विवरण
यह एक **साधारण वेब ऐप्लिकेशन** है जो **इंदौर, मध्य प्रदेश** की सभी सरकारी सेवाओं की जानकारी प्रदान करता है। यह **Cyber Café** में उपयोग के लिए आदर्श है।

## विशेषताएं
✅ **40+ सरकारी सेवाएं** - सभी महत्वपूर्ण सेवाएं एक जगह
✅ **सर्च फंक्शनलिटी** - PAN, आधार, बिजली आदि से सर्च करें
✅ **कैटेगरी फिल्टर** - दस्तावेज़, बैंकिंग, यूटिलिटी आदि
✅ **शुल्क जानकारी** - प्रत्येक सेवा की फीस दिखाई जाती है
✅ **रेस्पॉन्सिव डिजाइन** - मोबाइल, टैबलेट और डेस्कटप पर काम करता है
✅ **सरल इंटरफेस** - आसान नेविगेशन

## फाइलें
- `index.html` - मुख्य HTML स्ट्रक्चर
- `style.css` - स्टाइलिंग और डिजाइन
- `script.js` - सभी फंक्शनलिटी (सर्च, फिल्टर, आदि)

## उपयोग
1. **index.html** को किसी भी ब्राउजर में खोलें
2. सेवाएं खोजें और देखें
3. कैटेगरी से फिल्टर करें
4. पसंदीदा सेवाओं को मार्क करें

## सेवाएं शामिल
- आधार कार्ड
- पैन कार्ड
- पासपोर्ट
- ड्राइविंग लाइसेंस
- वाहन पंजीयन
- आयकर
- GST
- MSME
- और भी बहुत कुछ!

## तकनीक
- HTML5
- CSS3
- Vanilla JavaScript

## स्थानीय रूप से चलाएं
```bash
# सीधे index.html खोलें या
cd arun-mp-online-csc
open index.html
```
